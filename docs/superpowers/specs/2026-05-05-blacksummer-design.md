# Australia's Black Summer — Data Visualisation 2 Design Spec

**Date:** 2026-05-05  
**Course:** FIT2179 Data Visualisation 2, Monash University  
**Due:** 29 May 2026

---

## Domain, Why, Who

- **Domain:** Australian Bushfires — the 2019–2020 "Black Summer" season  
- **Why:** The Black Summer was Australia's most catastrophic fire season on record. Using satellite data, this visualisation reveals the true scale, geographic spread, and temporal escalation of the crisis in a way that raw numbers alone cannot.  
- **Who:** Average Australian — accessible language, no statistics jargon, storytelling focus.  
- **Distinctly different from DV1** (coffee).

---

## Data Sources (two required)

1. **NASA FIRMS MODIS satellite detections** (via Kaggle, "Australian Bush fire satellite data (NASA)" + "Fires from Space Australia")  
   - Files: `fire_archive_M6_*.csv`, `fire_nrt_M6_*.csv`  
   - Coverage: August 2019 – January 2020  
   - Fields: `latitude`, `longitude`, `brightness`, `frp`, `acq_date`, `confidence`, `daynight`, `type`  
   - Total combined: **199,417 fire detection points**

2. **WA Government Bush Fire Prone Areas 2025** (`package_show.json` → GeoJSON endpoint) + **data.gov.au 2019-20 Financial Year Bushfire Boundaries** (`package_show_3.json`)  
   - Used for the Chapter 4 "today's risk zones" overlay map

## Visual Style

- **Theme:** Smoke Ash — dark navy (`#111827`) background, warm sand (`#e8c9a0`) text, deep crimson (`#c0392b`) / flame red (`#e74c3c`) data colours  
- **Typography:** Georgia (serif) for headings/narrative; Courier New (monospace) for numbers/labels; sans-serif for axis labels  
- **Layout:** Single scrollable page, full-width, column-based sections

---

## Story Structure & Visualisations (12 total)

### Hero
- Animated stat counters: 199,417 detections · 6 months · 6 states · 11,164 MW peak FRP

### Chapter 1 — Where?
1. **Interactive dot map** — 500-sample fire points on Australia SVG, colour by month, size by FRP. Hover state highlights state and filters sidebar.
2. **Horizontal bar chart** — fire detections by state (NSW 101,569 dominates)

### Chapter 2 — When?
3. **Time-slider map** — Vega-Lite map with month filter slider; fire density updates in real time
4. **Monthly small multiples** — 5 mini maps (Aug → Jan), heat blob sized by detection count; shows the Dec explosion

### Chapter 3 — How Intense?
5. **FRP histogram** — distribution of fire radiative power (MW), log-scaled bins
6. **Month × State heat matrix** — 5×4 grid, cell colour = detection count
7. **Day vs Night bar** — 63% day / 37% night, with annotation
8. **Time-series line chart** — daily fire detection count Aug–Jan, annotated with key events

### Chapter 4 — Today
9. **Choropleth map** — 2019-20 official fire boundary polygons coloured by area burned
10. **Overlay map** — WA 2025 fire-prone zones overlaid on historical hotspot density
11. **Bubble chart** — state × month, bubble size = total FRP, interactive tooltip

### Bonus (for HD)
12. **Calendar heatmap** — daily fire counts in a calendar grid, colour = intensity

## Interactivity Plan

- Month slider → updates dot map + time-series highlight
- State hover/click → filters bar chart + shows state stats card
- Chart tooltips on all marks
- Show/hide legend toggle

---

## Technical Stack

- **HTML/CSS:** Pure.css grid + custom CSS (smoke ash theme)
- **Charts:** Vega-Lite (all 12 visualisations)
- **Data:** Pre-aggregated CSV files (< 2 MB total) generated by Python script
- **Maps:** Vega-Lite `geoshape` with TopoJSON (Australia states from naturalearth)
- **Hosting:** GitHub Pages (single `index.html`)
